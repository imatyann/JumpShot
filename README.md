# JumpShot -MVP v0.1.0
アクションゲームをしながらマウスで撃ちまくるアクションシューティング

## 1.遊び方(MVP)
- A：左に移動
- D：右に移動
- space：ジャンプ
- 左クリック：カーソル方向に主人公が弾を発射
- 右クリック：カーソル方向に主人公がワープ弾を発射
- R：リトライ／Esc：ポーズ
- 勝利：鍵を持って扉に触れる／敗北：敵が主人公に触れる/落下する

## ２．仕様
- 敵は画面右端からランダムスポーンの自機狙い
- 特殊弾が壁/敵に着弾すると主人公をそこにワープさせる
- 鍵を獲得したのち、扉に触れるとクリア
